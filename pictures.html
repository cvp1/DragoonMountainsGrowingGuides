Adding photos Drop a new .jpeg or .jpg file into the pictures/ folder, then run python3 scripts/sync-pictures.py from the repository root. The script resizes the original, stubs a manifest entry, and regenerates this gallery + the homepage strip. Edit pictures-manifest.json to fill in captions, re-run the script, and commit.
